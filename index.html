PROGRAMMER - DESIGNER

This project was created using Unity during the Global Game Jam, with the theme of "Root" serving as the inspiration.

The game involves growing a tree by creating branches and roots.

The branches collect sunlight and clouds on the surface and consume water, while the roots collect nutrients and water underground and consume air.

To grow the branches and roots, you must manage resources such as water, sunlight, air, and nutrients.

On this project, I work as a programmer / designer, my duties included:

  • Programming the animations using DoTweens.
  • Programming the events, like an event that increase the spawn of clouds.
  • Designing the game loop.
