Square Roots ! - PROGRAMMER / DESIGNER (2023)

This project was created during the Global Game Jam and was developed using Unity, inspired by the theme "Root".

In the game, players grow a tree by creating branches and roots, which collect sunlight, clouds, nutrients, and water. Proper resource management is crucial for success.

As a programmer/designer on this project, my duties included programming animations for roots and branches using DoTweens and designing the game loop.
